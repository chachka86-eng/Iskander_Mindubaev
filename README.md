 Book Tracker - Трекер прочитанных книг

Автор: Миндубаев Искандер 

Описание программы

Book Tracker это графическое приложение для управления списком прочитанных книг. Программа позволяет добавлять книги, удалять их, фильтровать по жанру и количеству страниц, а также сохранять данные в JSON файл для последующей загрузки.

Выполненные требования

Функциональные требования

- Форма с полями: Название книги, Автор, Жанр, Количество страниц
- Кнопка Добавить книгу для добавления данных в таблицу или список
- Фильтрация по жанру
- Фильтрация по количеству страниц больше указанного числа
- Сохранение данных в JSON файл
- Загрузка данных из JSON файла

Проверка корректности ввода

- Поля не должны быть пустыми
- Количество страниц должно быть числом положительным целым

Основные возможности

- Добавление книги с указанием названия, автора, жанра и количества страниц
- Проверка корректности ввода
- Отображение списка книг в таблице
- Фильтрация по жанру частичное совпадение, регистронезависимо
- Фильтрация по количеству страниц показывает книги с количеством страниц больше указанного
- Сохранение всех книг в JSON файл
- Загрузка книг из JSON файла
- Удаление выбранной книги из списка

Требования для запуска

- Python 3.6 или выше
- Стандартные библиотеки Python: tkinter, json, os

Инструкция по запуску

1. Скачайте проект
2. Запустите программу

python main.py

3. Работа с программой

Заполните поля в верхней части окна
Нажмите кнопку Добавить книгу
Книга появится в таблице
Для фильтрации используйте поля в блоке Фильтрация
Кнопка Сохранить сохранит все книги в файл books.json
Кнопка Загрузить загрузит книги из файла
Выберите книгу в таблице и нажмите Удалить чтобы удалить её

Примеры использования

Пример 1: Добавление книги

Название: Война и мир
Автор: Лев Толстой
Жанр: Роман
Страницы: 1300

Пример 2: Фильтрация по жанру

Добавьте несколько книг
В поле Фильтр по жанру введите: роман
Таблица покажет только книги жанра Роман

Пример 3: Фильтрация по страницам

В поле Страниц больше введите: 300
Таблица покажет книги с количеством страниц больше 300

Пример 4: Сохранение и загрузка

Нажмите Сохранить
Закройте программу
Запустите программу снова
Нажмите Загрузить все книги восстановятся
